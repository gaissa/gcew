# gcew

Installation
-
<br>
1. Install and activate the plugin.
<br>
2. Set the options.
<br>
3. Add the preferred shortcodes.
<br>

Shortcodes
-
<br>
1. ```[events_list]```
<br>
    Add to any page or post.
<br>
<br>
2. ```[calendar height="<int>"]```
<br>
    Add to any page or post.
<br>
<br>
3. ```[search]```
<br>
    Add anywhere (text widget, page, post and such).
